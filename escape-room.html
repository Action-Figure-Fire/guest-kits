The Lighthouse Cipher

60-Minute Printable Escape Challenge

The lighthouse keeper vanished 50 years ago. He left behind 5 locked puzzles. Solve them all to discover what happened. 2-6 players.

📋 Host Setup (5 min)

Print all puzzle pages. Cut them out. Place them in 5 envelopes numbered 1-5. Players must solve #1 to get the code for #2, and so on.

You need: A pen per player, this printed kit, a phone timer set to 60:00.

Hint system: Each team gets 3 hints. They ask the host, who reads from the Hint Guide below.

🧩 Puzzle 1 — The Keeper's Journal

The Entry

"Day 147. The light failed again. I've hidden the repair manual where only the observant will find it. Count the windows facing east. Multiply by the year I arrived minus 1970. The answer opens envelope two."

Clue hidden in the room: A sketch of the lighthouse shows 4 east-facing windows. A plaque reads "Keeper Hammond, stationed 1973."

Solution: 4 × (1973 - 1970) = 4 × 3 = 12 → Opens Envelope 2

🧩 Puzzle 2 — The Signal Code

Morse Code Message

.... . .-.. .--. / -- . / - .... . / -.- . -.-- / .. ... / ..--- .....

Decoded: "HELP ME THE KEY IS 25"

Include a Morse code reference sheet for players. Solution: 25 → Opens Envelope 3

🧩 Puzzle 3 — The Map Fragment

Torn Map Pieces

Four torn pieces of a coastal map. When assembled correctly, they reveal a dotted path leading to an X. Next to the X: a grid cipher.

The grid has letters A-Z in a 5×5 grid (I/J share a cell). Coordinates spell: R-U-N

Solution: The coordinates translate to "RUN" — but the code is the numerical position: R=18, U=21, N=14 → 182114

Too long for a simple code, so take first digits: 1-2-1 → Opens Envelope 4

🧩 Puzzle 4 — The Locked Box

The Riddle

"I have cities but no houses. I have mountains but no trees. I have water but no fish. I have roads but no cars. What am I?"

Answer: A MAP → Count the letters = 3 → Opens Envelope 5

🧩 Puzzle 5 — The Final Revelation

The Keeper's Last Message

A letter written in alternating visible/invisible text (every other word is faded). Players must fill in the blanks using context:

"I ____ leave ____ lighthouse ____ the ____ knows ____ truth. ____ light ____ not ____ — it ____ hidden. ____ me ____ the ____ below."

Full text: "I must leave this lighthouse before the world knows the truth. The light did not fail — it was hidden. Find me in the cave below."

🏆 ESCAPED! The keeper faked the light failure to cover his escape through a cave beneath the lighthouse. He was smuggling supplies to a stranded ship. He was a hero, not a villain.

💡 Hint Guide (Host Only)

Puzzle 1 Hints

Hint 1: "Look at the sketch carefully — count only the EAST side."

Hint 2: "The plaque has a year. Subtract 1970."

Hint 3: "4 × 3 = ?"

Puzzle 2 Hints

Hint 1: "It's Morse code. Use the reference sheet."

Hint 2: "The slashes separate words."

Hint 3: "The last word is a number. Two digits."
